_Clasifiers_:

1. k-NN from scratch
2. Softmax Classifier
3. Support Vector Machine
4. Random Forest
5. Multi Layer Perceptron

_Web Gui_:

1. React + Ts
2. 28\*28 painting area for testing out models
3. Leaderbord for all models
4. CSV input for more data
5. Results paired with the painting 28\*28
