## What to build

A runnable version-one shell that starts from the repo root on Windows, brings up the backend and frontend, serves a visible app shell, and proves the startup and storage path work end to end.

## Acceptance criteria

- [ ] A root PowerShell command starts the backend and frontend for local development on Windows when Python and Node are installed.
- [ ] The frontend renders a minimal shell and consumes a backend health or status endpoint successfully.
- [ ] A smoke test or equivalent verification covers startup and basic app-to-backend connectivity.

## Blocked by

None - can start immediately
